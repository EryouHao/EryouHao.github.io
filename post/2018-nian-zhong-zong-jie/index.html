2018 年终总结

2019-01-14 / 1 min read

📃又到了一年一度的年终总结，这是参加工作的第一个完整年。

🥺这一年总体来说，滑水较多，产出较少，尤其是文字方面，可怜巴巴。读过不少文章，写过不少代码，需提升、待吸收的知识一把一把。

🏫毕业快两年了，这一年多来遇到了许多贵人，使我成长了很多，对事情的看法和对世界的认知也清晰了很多。单是技术方面也不像从前那么迷茫与焦虑，学会了循序渐进，稳扎稳打，时间总会给准备的人最好的答案。

说说这一年的经历吧，除了在公司做的工作以外，当然公司的工作也锻炼了很多其他的东西。

⏱有一部分时间里，在思考使用 Vue.js 在工程化的最佳实践，如何组织前端项目代码，如何组织组件，如何抽象，如何管理依赖等等。

⏱有一部分时间里，写了一个“烂尾”的 Vue.js 组件库：Live UI，实现了14个常用的 UI 组件，虽然部分组件可能存在 bug，但是每实现一个新组件，总能学到一些新东西也是极好的。对 Vue.js 的理解又加深了一些，能够思考如何提供 API，哪些 API 是这个组件最需要提供给用户使用的过程也是不错的。

⏱有一部分时间里，写了一个 VSCode 浅色主题：Brackets Light Pro，当然这个 Repo 始于 2017，截止此文写作时已经有了 12,456 installs 和 28,544 downloads，很早就把主力编辑器换成了 VSCode，自己又比较偏好浅色主题，尤其是我大部分编程时间都在白天，所以就写了这样一个主题，希望你也喜欢「比心」。

⏱有一部分时间里，写了一个静态博客写作客户端：Hve Notes 。是的，这篇文章的创作即使用此工具完成。这是一个可以使用 Markdown 来进行记录生活、心情、知识、笔记、创意的跨平台（Windows & MacOS）工具。这大概是目前为止对我来说实现程度最高的一个开源项目吧，写代码、写文档、设计 Logo、打包发版、写主题，处理反馈等等，在一个编程新手的阶段来体验与尝试自己感兴趣的东西，也算是一种幸福吧。

⏱有一部分时间里，学习和实践新技术。使用 TypeScript + Egg + Vue + TypeORM + PostgreSQL + Puppeteer 写了一个小全栈项目。从新知识的学习到写点实际的东西，从开发到部署上线，整套开发体验还算不错，对于工程化的实践又多了一些。另外将自己的获取和学习主渠道更多地向国外倾斜，Twitter，Youtube，Medium，还有其他地一些网站、个博等等，蛮不错。

⏱有一部分时间里，关注与练习设计。在推特上关注了 Steve Schoger，个人觉得他的一些 Tips 很简单也很实用。可能在这之后或之前，更加关心前端Web 设计。尤其是在一些 UI 设计并不重的场合，比如一些内部使用的中后台系统，去尝试和融入自己的一点点学习，甚至将 UI 设计师的设计稿进行二次设计然后进行“设计交流”（好像最后气氛变成了投票和战队，这不是我想要的），这些尝试个人感觉还是蛮好的，不断试错，不断提升。

接下来说说，对于刚刚到来的 2019 的一些想法。

🧐保持不断学习，保持好奇心。
✍️用心做好每一个个人项目，比如 Hve Notes
🗞努力提升英语水平，能够在第一次英文介绍自己的时候，说出 「Vue点儿JS」还是有点尴尬😅
💻更新自己的技术栈
👻结交新朋友
💏照顾好亲人和女票
🍲争取做好每一道菜，像写好每一行代码一样
🏃运动
